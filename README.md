Commit Convention

---

| 작성         | 설명         |
|------------|------------|
| feat: 내용   | 기능 추가      |
| fix: 내용    | 버그 및 오류 수정 |
| docs: 내용   | 문서 수정      |
| hotfix: 내용 | 긴급 수정      |
| chore: 내용  | 기타 수정      |

Branch Convention

---
| 작성             | 설명              |
|----------------|-----------------|
| main           | master 브랜치      |
| develop        | main 서브 브랜치     |
| feat/#이슈번호-기능명 | 기능 추가용 feat 브랜치 |
| fix/#이슈번호-브랜치명 | 기능 수정용 fix 브랜치  |
